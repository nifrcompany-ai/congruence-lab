## 📋 О проекте

**Congruence Lab** — это локальная исследовательская платформа, созданная для конференции "Инженеры будущего". Приложение исследует цифровую идентичность через интерактивные эксперименты и психологические практики.

### Ключевая идея
**DSCI (Digital Self Congruity Index)** = 100 − (|Real−Digital| + |Ideal−Digital|) / 2

Где:
- **Real** — настоящее «я» (переживания, эмоции)
- **Ideal** — идеальное «я» (мечты, ценности)
- **Digital** — публичное «я» (соцсети, посты, образ)

## 🎯 Основные функции

### 📊 Тестирование
- **Быстрый тест** — 12 вопросов, 5–7 минут → архетип + DSCI
- **Полный тест** — 42 вопроса + контроль согласованности → 7-дневный план

### 📚 Библиотека
- Архетипы (12 типов цифровой идентичности)
- Научная база (Роджерс, Гоффман, Хиггинс, Райан-Деси, Тёркл)
- Практики, упражнения, медитации
- Статьи и книги по теме

### 💬 AI-Коуч (DEMO)
- 200+ шаблонов ответов по ключевым словам
- История чата (100 сообщений в IndexedDB)
- Анализатор постов с оценкой по 6 осям
- Реальный режим требует нейросети + согласие пользователя

### 🧪 Эксперименты
- Генератор постов по 12 маскам (Перфекционист, Уязвимый мудрец, Циник и т.д.)
- A/B-тестирование с эмуляцией реакций (лайки, комментарии, ER%)
- История экспериментов

### 📔 Дневник
- Ежедневная запись: аутентичность (1–10), эмоция, заметка
- Тепловая карта 30 дней
- График тренда
- Экспорт (JSON/CSV/PDF) с анонимизацией

### 👥 Команда
- Создание групп с 6-символьным кодом
- Анонимные снимки профилей участников
- Агрегированный радар-график
- Экпорт групповой аналитики

### 🎯 Челленджи
- 7-дневная программа базовой трансформации
- 30-дневная полная программа
- 30 дневных задач по 6 осям
- Web Notifications напоминания

### 🏆 Достижения
- Система уровней 1–10 от активности
- 15+ достижений разблокировки
- Звуковые уведомления
- Конфетти анимация

### ⚙️ Настройки
- Темы: dark (основная) / light
- Языки: русский / английский
- Масштабирование UI: normal / comfort / compact
- Экспорт/импорт данных
- Безопасное удаление всех данных

- ✅ **Данные не уходят** — полный контроль пользователя

## 🔧 Настройка и разработка

### Изменение логики приложения
1. Найдите нужный JS-файл в `/js/`
2. Отредактируйте функции
3. Сохраните, перезагрузите браузер (Ctrl+F5 — очистить кэш)

### Добавление новых вопросов теста
```javascript
// data/questions-short.js
global.CLQuestionsShort = [
  { id: '...', text: '...' },
  // ... добавьте новый
];
```

### Изменение дизайна
1. Отредактируйте `style.css`
2. Используйте CSS переменные для тем: `var(--primary)`, `var(--text)` и т.д.

### Локализация
1. Добавьте ключ в `locales/ru.json` и `locales/en.json`
2. В HTML: `<span data-i18n="new_key">Fallback text</span>`
3. JS автоматически подставит перевод
